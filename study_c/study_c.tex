\section{编译过程}
初学者在编译的时候最好一步一步来,以hello.c为例：

先将源文件编译成目标文件：gcc - c  hello.c
上一步生成hello.o文件，再将目标文件编译成可执行文件：gcc -o  hello  hello.o
生成的可执行文件是 当前路径下 a.out

最简单的是
gcc hello.c,直接生成a.out
